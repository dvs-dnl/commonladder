SCREENSHOT PLACEHOLDER
======================
WordPress requires a screenshot.png file (1200x900px, PNG) in the theme root
to display a preview thumbnail in Appearance > Themes.

To create it:
1. Install the theme on a local WordPress instance (Local by Flywheel, etc.)
2. Visit the homepage
3. Take a full-page screenshot using a browser tool or extension
4. Crop/resize to exactly 1200x900px
5. Save as screenshot.png in this directory
6. Delete this placeholder file

Design tip: The screenshot should show the header, hero section, and at least
the top of the category cards to represent the brand identity.

Brand colors for reference:
  Ladder Blue:   #1C3D6E
  Rung Amber:    #E8911A
  Resource Sage: #4A9E82
  Commons White: #F8F7F4
  Ground Dark:   #1A1A2E
