ID & Documents

ID & Documents resources near you

Help getting a government-issued photo ID, birth certificate, Social Security card, and other documents — a key first step for housing and employment.

How to use this directory

Getting an ID is often the first barrier people face when trying to access other services. Many programs listed here can help you get a state ID even if you don't have a birth certificate yet — they'll help you work through the process in order. Bring whatever documents you have and let staff guide you through next steps.

Free tool

Not sure which document to get first?

The ID Recovery Wizard maps the right order for your situation — birth certificate, Social Security card, state ID — and points you to the Arizona programs that recover them at no cost.
